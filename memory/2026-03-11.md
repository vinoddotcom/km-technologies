# 2026-03-11

- Investigated Telegram bot group messaging issue.
- User reported that bot does not respond in groups despite privacy mode being disabled.
- Checked logs and `openclaw.json` config.
- `openclaw.json` shows `channels.telegram.groupPolicy: "allowlist"`.
- Logs show `[module:telegram-auto-reply]` skipping group messages with `reason: "not-allowed"`.
- Determined that Telegram group messaging is currently restricted because the group ID is not in the `allowlist` (or no `allowFrom` group list is defined to permit group activity).
- Identified that the user attempted to use an invalid config key `allowGroups` which caused gateway restart loops.
